# oOh!media Investor Chat

A grounded chatbot that answers investor questions about oOh!media using its public investor materials and OML market data.

See [ARCHITECTURE.md](ARCHITECTURE.md) for the design and [IMPLEMENTATION-PLAN.md](IMPLEMENTATION-PLAN.md) for the build plan.

## Quick start

```bash
uv venv --python 3.11
source .venv/bin/activate
cp .env.example .env  # then fill in keys
```

Run instructions, Claude Desktop setup, and architecture summary land here as the build progresses.
